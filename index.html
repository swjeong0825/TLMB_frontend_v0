Tennis League Management Bot

Vanilla client for the chat-to-intent server. Open a league-specific URL to start chatting.

Chat API base URL is configured in js/config.js.

Pages

  • Create league
    /create-league/
  • Find league
    /find-league/
  • Find league (URL prefix only)
    /find-league-prefix/?prefix=MTB
  • Demo
    /demo
  • Player
    /league?league_id={league_id}
  • Admin
    /league?league_id={league_id}&host_token={host_token}
